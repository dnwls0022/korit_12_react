```tsx
import { AppBar, Container, Toolbar, Typography, CssBaseline } from '@mui/material'


import './App.css'

function App() {
  

  return (
    <Container maxWidth='xl'>
    <CssBaseline/>
    <AppBar position='static'>
      <Toolbar>
        <Typography variant='h6'>
          car shop
        </Typography>
      </Toolbar>
    </AppBar>
    
    
    </Container>
  )
}

export default App

```
다양한 코드에서 확인했듯이 MUI Contatiner 컴포넌트 내에 AppBar 를 하위컴포넌트로 위치시켯음. position prop을 이용해서 AppBar의 위치를 static으로 잡았다.
- static : 스크롤할때 앱 바가 상단에 고정되 =않고 사라짐
- fixed : 스크롤해서 넘어가더라도 앱 바가 상단에있음

- maxwidth : 높이최대

- cssBaseLine: 컴포넌트를 이용하여 여러 브라우저에서 리액트 앱의 형태가 균일하게 유지되도록 설정했음 일반적으로 최상단에 위치시켜 하위 컴포넌트들이 동일한 영향을 받도록 사용.

- 여기서 AppBar 의 PosiTion을 속성이라하지않고 프롭이라고 한 부분에 주목. html속성은 attribute였다
